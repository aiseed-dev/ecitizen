X-13ARIMA-SEATS のインストール (Linux)

X-13ARIMA-SEATS はセンサス局が Linux/Unix 向けにビルド済みの実行ファイルと ソースコードの両方を配布しています。コマンド1つで動く単体の実行ファイルなので、 インストールというより「置くだけ」です。ここでは Ubuntu/Debian 系を例にしますが、 Chromebook の Linux 環境 (Crostini) でも同じ手順で使えます。

プログラムには出力形式の違いで2種類あります。用途に合わせてどちらかを選びます (両方入れても構いません)。

実行ファイル出力向いている用途
x13as_htmlHTML ファイルブラウザで結果を読む。診断の表が見やすい
x13as_asciiテキストファイルスクリプトからの利用、R/Python 連携 (statsmodels は ASCII 版を使う)

1. ビルド済みバイナリを使う (最も簡単)

# ダウンロードして展開 (HTML出力版。最新版は census.gov で確認)
wget https://www2.census.gov/software/x-13arima-seats/x13as/unix-linux/program-archives/x13as_html-v1-1-b62.tar.gz
tar xf x13as_html-v1-1-b62.tar.gz

# 実行ファイルをパスの通った場所へ
sudo install -m 755 x13as/x13as_html /usr/local/bin/

# 動作確認 (引数なしで実行すると使い方が表示される)
x13as_html

ASCII 出力版は URL とファイル名の html を ascii に 読み替えてください (x13as_ascii-v1-1-b62.tar.gz)。 バージョンが上がるとファイル名の v1-1-b62 の部分が変わるので、 配布ページで最新版を確認してください。

2. ソースからビルドする

ビルド済みバイナリが動かない環境や、コンパイルオプションを変えたい場合は ソースからビルドします。Fortran コンパイラ (gfortran) を使います。 

sudo apt install gfortran make

wget https://www2.census.gov/software/x-13arima-seats/x13as/unix-linux/program-archives/x13as_htmlsrc-v1-1-b62.tar.gz
tar xf x13as_htmlsrc-v1-1-b62.tar.gz
cd x13as_htmlsrc-v1-1-b62

# gfortran 用の makefile が同梱されている
make -f makefile.gf

sudo install -m 755 x13as_html /usr/local/bin/

3. R から使う (導入はこれが一番簡単)

R の seasonal パッケージを入れると、依存パッケージの x13binary が X-13ARIMA-SEATS の バイナリごとインストールされるので、本体を別途用意する必要がありません。 

install.packages("seasonal")

library(seasonal)
m <- seas(AirPassengers)   # 付属のサンプルデータで季節調整
summary(m)
plot(m)
final(m)                    # 季節調整済み系列

4. Python から使う

statsmodels の x13_arima_analysis() が X-13ARIMA-SEATS のラッパーです。 本体 (ASCII 出力版) を上の手順で入れておき、実行ファイルの場所を 環境変数 X13PATH か引数 x12path で渡します。 

import pandas as pd
from statsmodels.tsa.x13 import x13_arima_analysis

series = pd.read_csv("sales.csv", index_col=0, parse_dates=True)["売上"]
res = x13_arima_analysis(series, x12path="/usr/local/bin")
print(res.seasadj)   # 季節調整済み系列

※ statsmodels は実行ファイル名 x13as を探すため、 sudo ln -s /usr/local/bin/x13as_ascii /usr/local/bin/x13as のようにシンボリックリンクを作っておくと確実です。

インストールできたら、X-13ARIMA-SEATS の使い方に進んでください。
